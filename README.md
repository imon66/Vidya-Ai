# InterviewAce - AI Interview Preparation Platform

A Next.js application for AI-powered interview practice with Groq and Gemini models.

## Features

- AI Interview Practice
- Coding Challenges
- Resume Review
- Progress Tracking
- Mobile Responsive Design

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Deploy (no environment variables needed)

Built with Next.js, TypeScript, and Tailwind CSS.
